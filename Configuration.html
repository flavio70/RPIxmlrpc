LAN Settings (eth0)

$ sudo nano /etc/network/interfaces

  • method 1

auto eth0

iface eth0 inet manual



set address from GUI (Network preferences) i.e:

IP: 151.98.130.155/24

ROUTER: 151.98.130.1

DNS SERVES: 135.239.25.53 135.120.0.18

  • method 2 (we are using this one)

$ sudo nano /etc/network/interfaces

auto eth0

iface eth0 inet static

address 151.98.130.155

netmask 255.255.255.0

gateway 151.98.130.1

dns-nameservers 135.239.25.53 135.120.0.18

restart network service

$ /etc/init.d/networking restart



NTP Settings

- add servers:

server 135.239.25.53 iburst

- restart ntp service

$ sudo /etc/init/d/ntp restart

- check ntp status

$ ntpq -pn



HOSTNAME Settings

$ sudo nano /etc/hostname

Insert desired hostname (RPI-PwrMan-xx) and close this file (CTRL+X...) and save changes (...press "y" and Enter). Now edit the second:

$ sudo nano /etc/hosts

And replace all items with previous hostname. Now close this file (CTRL+X...) and save changes (...press "y" and Enter). New hostname will be set after next reboot.

$ reboot



VNC Server Settings

$ apt-get install tightvncserver

$ vncserver :1 -geometry 1280x800

$ vncpasswd (set smoptics)

add vnc server as service

$ sudo nano /etc/init.d/vncboot as new file and add the following lines:

-------------------------------------------------------------------------------------------------------

# Default-Start: 2 3 4 5

# Default-Stop: 0 1 6

# Short-Description: Start VNC Server at boot time

# Description: Start VNC Server at boot time.

### END INIT INFO



USER=pi

HOME=/home/pi



export USER HOME



case "$1" in

start)

echo "Starting VNC Server"

#Insert your favoured settings for a VNC session

su - $USER -c "/usr/bin/vncserver :1 -geometry 1280x800 "

;;



stop)

echo "Stopping VNC Server"

/usr/bin/vncserver -kill :1

;;



*)

echo "Usage: /etc/init.d/vncboot {start|stop}"

exit 1

;;

esac



exit 0

-------------------------------------------------------------------------------------------------------

$ sudo chmod +x /etc/init.d/vncboot

$ sudo systemctl enable vncboot

$ sudo service vncboot start|stop
